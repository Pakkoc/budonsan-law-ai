```
├── 홈 (/)

├── 질문 상세 (/questions/:id)
├── 검색 & 목록 (/search)
```

## 2. 사용자 흐름 (사용자 흐름)

### **핵심 작업 1: 일반인 질문 등록 및 답변 확인**
1. 홈페이지 방문 → 상단 검색창에 간단 질문 입력
2. "질문하기" 버튼 클릭 → 회원가입/로그인 유도
3. 카테고리 선택 (전세/임대차, 매매, 세금, 대출 등)
4. 개인 상황 세부 입력 폼 작성
5. 질문 제출 → 질문 상세 페이지로 이동
6. AI 실시간 답변 확인 (3초 내 생성)
7. 변호사 답변 대기 → 알림 수신 시 답변 확인
8. 답변 북마크 저장 또는 PDF 출력

### **핵심 작업 2: 변호사 답변 작성 및 수익 창출**
1. 변호사 회원가입 → 자격증 업로드
2. 관리자 승인 대기 → 승인 완료 알림
3. 변호사 대시보드 접근 → 관심 키워드 설정
4. 새로운 질문 웹푸시 알림 수신
5. 답변 대기 질문 목록에서 관심 질문 선택
6. 답변 작성 → 1,000원 차감 안내 확인
7. 답변 제출 → 질문 페이지에 즉시 노출
8. 프로필 조회수 및 연락처 노출을 통한 리드 확보

### **핵심 작업 3: 관리자 법령 업데이트 및 변호사 승인**
1. 관리자 로그인 → 관리자 대시보드 접근
2. 신규 법령 PDF 업로드 → 중복 검사 통과
3. 버전 관리 시스템에 자동 등록
4. 변호사 인증 대기 목록 확인
5. 제출된 서류 검토 → 승인/거절 처리
6. 신고된 답변 검토 → 필요시 삭제 처리

## 3. 네비게이션 구조 (네비게이션 구조)

### **글로벌 네비게이션 (GNB)**
- **위치**: 상단 고정 (Sticky Header)
- **구성 요소**:
  - 좌측: 로고 (홈으로 이동)
  - 중앙: 통합 검색창
  - 우측: 질문하기 버튼, 로그인/프로필 메뉴

### **메인 네비게이션**
- **홈**: 최신 질문, 인기 질문, 카테고리 탭
- **질문하기**: 단계별 진행 표시기 (Stepper)
- **검색/목록**: 좌측 필터 사이드바, 정렬 옵션

### **사용자별 네비게이션**
- **일반 사용자**: 마이페이지 (내 질문, 북마크, 설정)
- **변호사**: 변호사 대시보드 (답변 관리, 잔액, 프로필)
- **관리자**: 관리자 패널 (승인, PDF 관리, 통계)

### **푸터 네비게이션**
- 이용약관, 개인정보처리방침, 고객센터, 회사 정보

## 7. URL 구조 (URL 구조)

### **일반 규칙**
- 소문자 사용, 단어 구분은 하이픈(-)
- 복수형 명사 사용 (questions, lawyers)
- RESTful 패턴 준수

### **주요 URL 패턴**
- **홈**: `/`
- **질문 목록**: `/questions`
- **질문 상세**: `/questions/:id`
- **카테고리별**: `/category/:slug`
- **검색**: `/search?q=keyword&category=tax`
- **사용자**: `/my/:section`
- **변호사**: `/lawyer/:section`
- **관리자**: `/admin/:section`

### **SEO 최적화**
- 의미있는 URL 슬러그 사용
- 카테고리명은 한글 → 영문 변환 (전세임대차 → rental)
- 질문 ID는 숫자 기반 자동 증가

## 8. 컴포넌트 계층 구조 (컴포넌트 계층 구조)

### **글로벌 컴포넌트**
- **Header**: 로고, 검색창, 네비게이션, 사용자 메뉴
- **Footer**: 링크, 저작권, 연락처 정보
- **Sidebar**: 카테고리 필터, 광고 슬롯
- **Modal**: 로그인, 회원가입, 확인 대화상자

### **페이지별 컴포넌트**
#### 홈페이지
- **HeroSection**: 메인 배너, 검색창
- **CategoryTabs**: 카테고리 탭 네비게이션
- **QuestionCard**: 질문 미리보기 카드
- **StatsWidget**: 서비스 통계 표시

#### 질문 관련
- **QuestionForm**: 질문 등록 폼
- **AIAnswerCard**: AI 답변 표시 카드
- **LawyerAnswerCard**: 변호사 답변 카드
- **RelatedQuestions**: 유사 질문 추천 목록

#### 사용자 인터페이스
- **UserProfile**: 사용자 프로필 정보
- **ActivityStats**: 활동 통계 차트
- **BookmarkList**: 북마크 목록
- **NotificationBell**: 알림 아이콘

#### 변호사 전용
- **LawyerDashboard**: 변호사 대시보드 레이아웃
- **PendingQuestions**: 답변 대기 질문 목록
- **BalanceWidget**: 잔액 표시 위젯
- **AnswerEditor**: 답변 작성 에디터

#### 관리자 전용
- **AdminPanel**: 관리자 패널 레이아웃
- **ApprovalQueue**: 승인 대기 목록
- **DocumentUpload**: PDF 업로드 컴포넌트
- **AnalyticsDashboard**: 분석 대시보드

### **공통 UI 컴포넌트**
- **Button**: Primary, Secondary, Danger 변형
- **Input**: 텍스트, 이메일, 비밀번호 필드
- **Card**: 기본 카드, 답변 카드, 통계 카드
- **Badge**: 상태 표시, 카테고리 태그
- **Loading**: 스켈레톤 UI, 스피너
- **Alert**: 성공, 오류, 경고 메시지
- **Tooltip**: 도움말 툴팁
- **Pagination**: 페이지 네비게이션

### **상태 관리 컴포넌트**
- **AuthProvider**: 인증 상태 관리
- **ThemeProvider**: 다크/라이트 테마
- **NotificationProvider**: 알림 시스템
- **SearchProvider**: 검색 상태 관리
